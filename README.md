# DEV zen mode

This is a userscript that adds a keyboard shortcut (Shift+Z) to toggle top,
bottom and sidebars on [DEV][dev]. This improves the reading experience with
long articles.

More details in the [blog][blog] or on [DEV][dev-blog].
